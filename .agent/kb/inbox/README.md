# kb inbox — draft cards awaiting promotion

An unattended agent drops a DRAFT knowledge card HERE — it never edits the live kb, which would be
a self-modifying prompt. Inbox cards are committed but NEVER loaded into any prompt.

A human, or an explicitly-invoked review, promotes a card by moving it up into `../` (kb/), stamping
its `verified:` date, and adding its line to `../README.md`'s index. Empty is the normal state.
